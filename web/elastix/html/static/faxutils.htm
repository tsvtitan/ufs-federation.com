JHylaFAX
JHylafax y a Hylafax frontend written in Java. It can be installed in Linux, Windows and MacOS.
Manufacturer Description
"A Java client licensed under the GPL for the HylaFAX fax server. JHylaFAX is purely written in Java and runs on any Java 6.0 enabled platform. It has a small footprint, starts up fast and features a sleek user interface."
Download Link: http://sourceforge.net/project/showfiles.php?group_id=136485
Manufacturer: JHylaFAX
 
Winprint HylaFAX
Winprint HylaFAX is a utility created to send faxes from any Windows machine through a HylaFAX Server.
Manufacturer Description
"Winprint HylaFAX is a Windows print monitor designed to send its output directly to a HylaFAX Server. Once installed, you can print to a HylaFAX Server from any application, and it will pop up a simple dialog box, allowing you to enter the destination FAX number. It's not intended to be fancy or all encompassing, just a quick and easy way to send FAXes from any Windows application."
Download Link: http://sourceforge.net/project/showfiles.php?group_id=91606
Manufacturer: Winprint Hylafax
